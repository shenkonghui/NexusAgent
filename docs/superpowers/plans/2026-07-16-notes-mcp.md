# 笔记 MCP Server 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 openNexus 暴露只读 Notes MCP（HTTP `/mcp/notes`），用一次性 Token 鉴权；本系统 Agent 经 ACP `McpServers` 注入，外部客户端可在「笔记设置」复制 Endpoint/Token 连接。

**Architecture:** 进程内用官方 `modelcontextprotocol/go-sdk` 挂载 Streamable HTTP；Token 存 `NoteSettings.mcp_token`（明文可回显、只生成一次）；工具只读调用现有 `NoteRepository`；`Connection.NewSession` 按用户注入 HTTP MCP。

**Tech Stack:** Go、Gin、GORM、`github.com/modelcontextprotocol/go-sdk/mcp`、coder/acp-go-sdk、React/TS 设置页

**对应规格：** `docs/superpowers/specs/2026-07-16-notes-mcp-design.md`

**实现状态：** Task 1–5 已完成；提交待用户确认。

## Global Constraints

- 只读：仅 `list_note_tags` / `list_notes` / `get_note`
- Token 只生成一次，不轮换、不作废；空 Token = 未启用
- MCP 鉴权用 `Authorization: Bearer <mcp_token>`，与 JWT 分离
- `list_notes`：有有效标题时不返回正文；`title == ""` 或 `title == "无标题"` 时返回完整 `content`（当前业务会把无标题写成「无标题」）
- 新方法 ≤50 行；修改函数若超 100 行需拆分；尽量少改原有逻辑
- 文档与 UI 文案使用中文
- 提交仅在用户明确要求时执行（本计划步骤中的 Commit 可跳过或攒到用户要求时再做）

```go
var ErrMCPTokenAlreadySet = errors.New("mcp token 已存在")
var ErrMCPTokenNotFound = errors.New("mcp token 无效")

func (r *NoteSettingsRepository) FindByMCPToken(token string) (*models.NoteSettings, error) {
	token = strings.TrimSpace(token)
	if token == "" {
		return nil, ErrMCPTokenNotFound
	}
	var s models.NoteSettings
	err := r.db.Where("mcp_token = ?", token).First(&s).Error
	if errors.Is(err, gorm.ErrRecordNotFound) {
		return nil, ErrMCPTokenNotFound
	}
	if err != nil {
		return nil, err
	}
	return &s, nil
}

func (r *NoteSettingsRepository) SetMCPTokenOnce(userID uint, token string) error {
	s, err := r.FindByUserID(userID)
	if err != nil {
		return err
	}
	if strings.TrimSpace(s.McpToken) != "" {
		return ErrMCPTokenAlreadySet
	}
	if s.ID == 0 {
		s.UserID = userID
		s.McpToken = token
		return r.db.Create(s).Error
	}
	s.McpToken = token
	return r.db.Save(s).Error
}
```

修改 `Upsert`：在 `Save` 前保留 `existing.McpToken`（与 `ClassifySessionID` 同样处理）：

```go
s.McpToken = existing.McpToken
```

```go
// GenerateMCPToken POST /api/v1/notes/settings/mcp-token
func (h *NoteHandler) GenerateMCPToken(c *gin.Context) {
	uid, ok := currentUserID(c)
	if !ok {
		Fail(c, http.StatusUnauthorized, "UNAUTHORIZED", "未认证")
		return
	}
	b := make([]byte, 32)
	if _, err := rand.Read(b); err != nil {
		Fail(c, http.StatusInternalServerError, "INTERNAL", "生成 token 失败")
		return
	}
	token := hex.EncodeToString(b)
	if err := h.settingsRepo.SetMCPTokenOnce(uid, token); err != nil {
		if errors.Is(err, repository.ErrMCPTokenAlreadySet) {
			Fail(c, http.StatusConflict, "MCP_TOKEN_EXISTS", "MCP Token 已生成，不可重复生成")
			return
		}
		Fail(c, http.StatusInternalServerError, "INTERNAL", "保存 MCP Token 失败")
		return
	}
	Success(c, http.StatusOK, gin.H{"mcp_token": token})
}
```

imports：`crypto/rand`、`encoding/hex`。

```go
package notesmcp

// BearerUserID 从 Authorization 解析 mcp token 并返回 userID。
func BearerUserID(r *http.Request, settings *repository.NoteSettingsRepository) (uint, error) {
	h := r.Header.Get("Authorization")
	if !strings.HasPrefix(h, "Bearer ") {
		return 0, errors.New("unauthorized")
	}
	tok := strings.TrimSpace(strings.TrimPrefix(h, "Bearer "))
	s, err := settings.FindByMCPToken(tok)
	if err != nil {
		return 0, err
	}
	return s.UserID, nil
}
```

```go
srv := mcp.NewServer(&mcp.Implementation{Name: "opennexus-notes", Version: "1.0.0"}, nil)

// list_note_tags: 无参 → JSON 字符串数组
```

Tool handler 内从 `context`/`request` 取 `userID`（由外层 HTTP 中间件写入 `context.WithValue`）。

挂载示例：

```go
func Handler(noteRepo *repository.NoteRepository, settingsRepo *repository.NoteSettingsRepository) http.Handler {
	inner := mcp.NewStreamableHTTPHandler(func(r *http.Request) *mcp.Server {
		return newServer(noteRepo) // tools 闭包读 userID from r.Context()
	}, &mcp.StreamableHTTPOptions{Stateless: true})

	return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
		uid, err := BearerUserID(r, settingsRepo)
		if err != nil {
			http.Error(w, "unauthorized", http.StatusUnauthorized)
			return
		}
		ctx := context.WithValue(r.Context(), userIDKey{}, uid)
		inner.ServeHTTP(w, r.WithContext(ctx))
	})
}
```

用 `gin.WrapH` 注册到 `r.Any("/mcp/notes", gin.WrapH(h))` 与 `r.Any("/mcp/notes/*path", gin.WrapH(h))`（按 SDK 路径需要二选一或都挂）。

```go
func (s *Service) notesMCPServers(userID uint) []acp.McpServer {
	if s.noteSettings == nil {
		return nil
	}
	st, err := s.noteSettings.FindByUserID(userID)
	if err != nil || strings.TrimSpace(st.McpToken) == "" {
		return nil
	}
	base := strings.TrimRight(s.publicBaseURL, "/")
	return []acp.McpServer{{
		Http: &acp.McpServerHttpInline{
			Name: "opennexus-notes",
			Type: "http",
			Url:  base + "/mcp/notes",
			Headers: []acp.HttpHeader{{
				Name:  "Authorization",
				Value: "Bearer " + st.McpToken,
			}},
		},
	}}
}
```
